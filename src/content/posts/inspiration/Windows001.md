## 常用快捷键

- 恢复已经关闭的网页 ：Ctrl + Shift + T
- 文件资源管理器：Win+E
- 现实桌面：Win+D
- 打开窗口菜单:Alt+Space [快捷键+M(移动) 可拖回屏幕外的窗口]

-   ncpa.cpl : 网络适配器
-   gpedit.msc : 配置本地策略组
-   inetcpl.cpl  : Internet Properties


## **一、效率快捷键：不碰鼠标也能飞快操作**

### **1. `Win + V`：召唤剪贴板历史**

- **操作方法**：首次使用需在提示中点击“开启”。之后按 `Win + V` 即可调出剪贴板面板，支持多条目记录、常用内容置顶、跨设备同步（需登录 Microsoft 账号）。
- **适用场景**：频繁复制粘贴文字、图片、代码片段，避免反复切换窗口。

### **2. `Win + Shift + S`：高级截图与标注**

- **操作方法**：按下后屏幕变暗，顶部出现截图工具栏（矩形/自由形状/窗口/全屏）。截图自动存入剪贴板，右下角弹出通知可快速进入“截图和草图”进行标注、保存。
- **适用场景**：工作汇报、教程制作、错误反馈截图。

### **3. `Win + Ctrl + Shift + B`：显卡驱动一键重置**

- **操作方法**：系统黑屏、画面卡顿或外接显示器无信号时，同时按下该组合键。屏幕会闪烁 1~2 秒并伴随“滴”声，显卡驱动将热重载。
- **注意**：仅重置显示驱动，不关闭任何程序。Win10 1809 及以上 / Win11 均支持。

## **二、系统优化与存储管理：告别 C 盘爆满与卡顿**

### **4. `cleanmgr`：专业级磁盘清理**

- **操作方法**：`Win + R` 输入 `cleanmgr` 回车 > 选择 C 盘 > 点击“清理系统文件” > 勾选 **Windows 更新清理**、**临时文件**、**回收站** 等 > 确定。
- **效果**：可安全释放 3GB~10GB 空间，不影响系统稳定性。建议每月执行一次。

### **5. 管理开机自启，加速启动**

- **操作方法**：`Ctrl + Shift + Esc` 打开任务管理器 > 切换至“启动应用”选项卡 > 右键非必要程序（如云盘、聊天软件、驱动助手）选择“禁用”。
- **提示**：保留杀毒软件、输入法、硬件控制中心即可。禁用后软件仍可随时手动启动。

### **6. 智能配置“存储感知”**

- **路径**：`设置 > 系统 > 存储 > 存储感知`
- **建议设置**：开启后设置为“每周”或“每月”运行，仅勾选“删除临时文件”与“清理回收站（超过 30 天）”。**务必取消勾选下载文件夹**，避免误删重要资料。

## **三、多任务与文件管理：像专业用户一样工作**

### **7. `Win + Ctrl + D`：虚拟桌面隔离工作流**

- **操作**：按 `Win + Ctrl + D` 新建桌面；`Win + Ctrl + 左/右` 快速切换；`Win + Tab` 查看全局任务视图，可拖拽窗口分配桌面，右键桌面可命名（如“办公”“设计”“娱乐”）。
- **优势**：会议、写代码、剪视频互不干扰，任务栏可设置为“仅显示当前桌面窗口”。

### **8. 批量重命名：告别手动逐改**

- **操作**：在资源管理器中选中多个文件 > 按 `F2` 输入基础名称（如 `项目资料`）回车 > 系统自动命名为 `项目资料(1)`、`项目资料(2)`…
- **技巧**：若需更复杂命名规则，可借助 PowerShell 或免费开源工具 PowerToys 的 `PowerRename`。

### **9. 快速访问“最近使用”文件**

- **操作**：打开资源管理器 (`Win + E`) > 左侧导航栏默认显示“快速访问”。右键“快速访问”可自定义是否显示“最近使用的文件”与“常用文件夹”。
- **提示**：开启后，双击即可秒开昨天编辑过的文档，无需记忆路径。

## **四、隐私安全与日常维护：干净、安全、省心**

### **10. 关闭个性化广告追踪**

- **路径**：`设置 > 隐私和安全性 > 常规`（Win10 为“隐私”）
- **操作**：关闭 **允许应用使用广告 ID 为我提供个性化广告**。同时可向下关闭“允许网站提供本地化内容”等选项，减少数据收集。

### **11. `Win + L`：1秒锁定电脑（离座必备）**

- **操作**：离开座位前按下 `Win + L`，系统立即锁屏并跳转至登录界面。配合 PIN 码、指纹或 Windows Hello，兼顾安全与便捷。
- **建议**：在 `设置 > 账户 > 登录选项` 中启用“需要 Windows Hello 登录”以强化保护。

### **12. 隐藏敏感文件夹**

- **操作**：右键目标文件夹 > `属性` > `常规` 选项卡勾选“隐藏” > 确定。
- **恢复查看**：资源管理器顶部点击“查看” > “显示” > 勾选“隐藏的项目”即可临时显示。适合存放私密文档或备用配置。

## **❓ 常见问题解答（FAQ）**

**Q：这些技巧需要管理员权限吗？会不会影响系统稳定性？**A：均为 Windows 官方内置功能，无需第三方权限。部分操作（如清理系统文件、修改启动项）会触发 UAC 提示，属正常安全机制。按文中步骤操作不会破坏系统。

**Q：Win10 和 Win11 的设置路径有差异吗？**A：核心快捷键完全一致。设置菜单名称略有不同（如 Win10 为“隐私”，Win11 为“隐私和安全性”），文中已标注兼容版本，按路径关键词搜索即可快速定位。

**Q：剪贴板历史或虚拟桌面会占用大量内存吗？**A：剪贴板历史默认仅保留最近 25 条文本/图片，内存占用极低；虚拟桌面仅为窗口分组逻辑，不复制进程，性能开销可忽略不计。

**Q：如何备份当前系统设置，防止误操作？**A：建议开启“文件历史记录”或定期创建系统还原点：`Win + R` 输入 `sysdm.cpl` > “系统保护”选项卡 > 选择 C 盘点击“创建”。
