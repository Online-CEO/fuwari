## 강의 소개

5주 과정의 바이브 코딩 1인 창업 부트캠프(AI Product Builder) 관련 링크와 자료를 정리한 글입니다.

## 커리큘럼 요약

### 1주차: 기획 → 첫 수익

- 개요 · 웹 개발 기초 · 배포 · API · 광고 수익화

### 2주차: 유입 → 성장

- 데이터 분석 · 퍼널 · SEO & GEO · 바이럴 구조 · 마케팅

### 3주차: AI → 결제

- 아이디어 기획 · React · Serverless · AI API 활용 · 바이브 디자인 · 글로벌 결제 구현

### 4주차: SaaS → 반복 매출

- 데이터베이스 · 회원가입/로그인 · 구독형 서비스 구현

### 5주차: 확장 → Exit

- 실무 운영 · 퍼포먼스 마케팅 · 앱 개발 · 미국 법인 · Stripe · Exit 전략
